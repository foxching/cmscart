
# CMS Shopping Cart

A simple node/express based  shopping cart. 


## Tech stack

* Nodejs & Express
* Ejs templating engine
* Mongodb
* ajax sessions, authentication, boostrap, 
and more

### Installation

Install all the dependencies

```sh
$ npm install
```
### Serve
To serve in the browser  

```sh
$ npm  start
```
